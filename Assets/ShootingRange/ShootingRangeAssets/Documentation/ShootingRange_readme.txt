**************************    DESCRIPTION   ******************************

Learn your characters how to shoot! This indoor shooting range
is ideal place for that. Five lanes and target distances from 5 to 25m.

Product includes shooting range hall with poseable walls, entrance
door, service gates and 5 targets.

Each target can move separately, and each one can use one of 4
material styles (different number and layout of bullet holes):
- Clean
- Enthusiast
- N00b
- Sharpshooter

Including 4 props:
- Handgun
- Ear protection phones (with morph for male/female characters)
- Chairs
- Empty gun shells on the floor

Light set is included for Poser version (DS Iray version use emissive
materials for light by default)

Notice: Daz Studio version works ONLY with Nvidia Iray materials!

Of course, you can use your own weapons ;)

*******************   SYSTEM REQUIREMENTS    **********************

A computer capable of running the appropriate version of Poser and DS

***********************   NECESSARY FILES   ***************************

This is a Stand Alone product.
 
**************************  INSTALATION   *******************************

Unpack Poser zip files to Your Poser Runtime folder. 

Unpack DS zip files to Your My DAZ 3D Library
